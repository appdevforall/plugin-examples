AI Assistant — Agent Guide

The Agent tab is a chat assistant that can read, search, and edit your project through an approval-gated tool loop. Inference is served by the companion AI Core plugin, so install AI Core first.

Choosing a backend

  • Local (on-device) — runs a .gguf model via llama.cpp. Open Settings, pick a model file from your Downloads folder. Nothing leaves the device.
  • Gemini (cloud) — enter a Gemini API key in Settings. Prompts and any file contents the agent reads are sent to Google over HTTPS.

What the agent can do

  • Read and search files within the current project.
  • Create and update files (asks for approval before writing).
  • Trigger a Gradle sync and read build output.
  • Add dependencies and generate code from templates.
File tools are confined to the project root: paths that resolve outside it are rejected, so the model cannot read or overwrite arbitrary files on the device.

Troubleshooting

  • "No model configured" — select a .gguf file (Local) or set an API key (Gemini) in Settings.
  • Gemini errors — check the API key, network connection, and quota.
  • Agent tab missing — confirm both AI Core and AI Assistant are installed and the IDE was restarted.
