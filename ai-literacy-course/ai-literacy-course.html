AI Literacy Course

An offline "Introduction to AI" course from Learn AI Anywhere, packaged for Code On The Go.

Executive overview

The AI Literacy Course plugin brings Learn AI Anywhere's complete Introduction to Artificial Intelligence course into Code On The Go, playable entirely offline. It bundles twenty-six short lesson videos, interactive HTML activities, the NeuroPocket machine-learning app, and teacher resources into a single full-screen experience. Once installed, it needs no internet connection — making it suitable for classrooms and communities with limited or no connectivity. It is aimed at learners and educators who want a structured, self-contained AI literacy curriculum on a device they already use for coding.

Core functionality

  • Offline video course — an introduction plus four lessons (26 videos total), presented in playback order and played inline with fullscreen support.
  • Interactive activities — self-check quizzes and worksheets (HTML), plus the NeuroPocket app, which runs an image classifier in the browser.
  • Teacher resources — a teacher guide, slide deck, and printable worksheets rendered as PDF.
  • Generated navigation — a lesson list is built automatically from the bundled content, so videos are reachable in a couple of taps.
  • Fully offline — the entire course ships inside the plugin and is served locally; nothing is uploaded and no learner data leaves the device.

Technical architecture

The plugin implements IPlugin, UIExtension, and DocumentationExtension. Its pieces:

ComponentResponsibility
AiLiteracyCoursePluginLifecycle, an editor-sidebar launcher (getSideMenuItems) that opens the course via IdeUIService.openPluginScreen, and tooltip/Tier-3 help.
CourseInstallerOne-time extraction of the bundled ZIP into IdeEnvironmentService.getPluginDataDirectory() using IdeArchiveService, including the nested activity zips.
CourseShellGenerates the navigation index.html over the extracted tree (the bundle ships no web shell of its own).
CourseFragmentFull-screen WebView host. Serves the extracted files through a WebViewAssetLoader custom PathHandler over a virtual https origin (required for video and the NeuroPocket service worker). Handles Back, rotation, and fullscreen video.

The ~110 MB course ZIP is not committed to source control; a Gradle downloadAssets task fetches it at build time (verifying a pinned MD5) and packs it into the .cgp. The only declared permission is ide.environment.write, needed for the one-time extraction into the plugin's data directory.

Usage

  1. Install the plugin: Preferences → Plugin Manager → +, then select ai-literacy-course.cgp and enable it.
  2. Open a project so the editor (and its sidebar) is visible.
  3. Tap the AI Literacy Course icon (graduation cap) in the editor sidebar.
  4. On first launch the course unpacks once (a brief progress screen); after that it opens instantly.
  5. Browse lessons, tap a video to play it (and the fullscreen control to enlarge), or open an activity or PDF. Use the device Back gesture to step back through the course and exit.

Key benefits

  • Works with no internet — ideal for offline-first classrooms and low-connectivity environments.
  • Self-contained — videos, activities, and resources travel inside the single plugin file.
  • Privacy-preserving — self-check activity scores stay on-device; nothing is transmitted.
  • Low friction — a generated lesson index puts the course content a couple of taps away, inside the IDE students already use.
