Keystore Generator

Generate Android .jks keystores for signing release builds — directly from Code On The Go.

Creating a keystore

  1. Open Generate Keystore from the sidebar, or the main editor tab.
  2. Fill in the keystore filename, passwords, alias, and certificate details.
  3. Tap Generate Keystore. The file is written to your project's app/ directory.

Certificate fields

CN
Common name — your name or organization.
OU
Organizational unit (optional).
O
Organization (optional).
L / ST / C
Locality, state, and two-letter country code.

Keep it safe

Losing your keystore means you cannot publish updates to a previously signed app. Back it up to at least two secure locations and never commit it to source control.
