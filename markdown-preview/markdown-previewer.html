1. Overview

A Code on the Go plugin for previewing Markdown and HTML files with live rendering. Files can be selected from the project tree or directly from device storage.

2. Features

Markdown preview

Full support for Markdown:

  • Headers, bold, italic, strikethrough
  • Code blocks with syntax-highlighting styles
  • Tables and task lists
  • Links and images
  • Blockquotes

HTML preview

Direct rendering of HTML files:

  • Automatic dark-mode support
  • Responsive layout

File selection

  • Select from the project file tree
  • Select from device storage (SD card, downloads, etc.)
  • Context-menu integration for supported files

3. Supported File Types

FormatExtensions
Markdown .md, .markdown, .mdown, .mkd, .mkdn
HTML .html, .htm, .xhtml

4. Building

cd markdown-previewer-plugin

# Build debug plugin
./gradlew assemblePluginDebug

# Build release plugin
./gradlew assemblePlugin

Output:

VariantPath
Debugbuild/plugin/markdown-previewer-debug.cgp
Releasebuild/plugin/markdown-previewer.cgp
Plugin permissions: filesystem.read only — the plugin reads source files but never writes back.
