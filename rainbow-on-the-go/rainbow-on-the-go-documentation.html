RainbowOnTheGo Plugin

Depth-colored parentheses, brackets, and braces for the Code On The Go editor.

1. Executive Overview

RainbowOnTheGo makes nested code easier to read by coloring each pair of brackets according to how deeply it is nested. An opening delimiter and its matching closing delimiter share the same color, so you can see at a glance which ( pairs with which ), even in deeply nested expressions.

The plugin computes the coloring itself and the Code On The Go editor merges it on top of the normal highlighting. It adds no toolbar buttons, requests no permissions, makes no network calls, and reads no files.

2. Core Functionality

The plugin colors three delimiter pairs — (), [] and {} — cycling through six colors by nesting depth:

DepthDay themeNight theme
0, 6, 12…RedRed
1, 7, 13…OrangeOrange
2, 8, 14…YellowYellow
3, 9, 15…GreenGreen
4, 10, 16…BlueBlue
5, 11, 17…PurplePurple

Brackets that appear inside string literals and comments are left alone, so a ( inside "text (here)" keeps its normal string color.

3. Technical Architecture

The plugin implements EditorDecorationProvider, a generic additive extension in the Code On The Go plugin API. Its only method, decorate(text, start, end, isDark), returns a list of foreground-color spans for a region of editor text; the IDE merges them on top of the existing highlighting. Because the only output is a list of colored ranges, a provider can never replace or suppress the editor's syntax highlighting — and the IDE has no knowledge of brackets, depth, or palettes. All of that lives in the plugin.

For each region, decorate scans the text for bracket characters, tracks the running nesting depth (recovered by scanning back from the region start), skips brackets inside strings and comments, and emits a span per bracket colored palette[depth % palette.size]. Detection is text-based, so it works in every language — not only those whose grammar tags brackets — and the same generic decoration hook can power any other plugin (indent guides, marker tinting, …).

4. Usage

  1. Install rainbow-on-the-go.cgp from the Code On The Go Plugin Manager.
  2. Enable the plugin. Open editors repaint automatically.
  3. Open any source file with nested brackets — each nesting level is a different color, and matching delimiters share a color.
  4. Switch between light and dark themes; the palette follows the theme.
  5. Disable the plugin to return brackets to the editor's normal color.

5. Key Benefits

  • Readability: matching delimiters are instantly identifiable in deeply nested code.
  • Theme-aware: separate palettes keep colors legible on both light and dark backgrounds.
  • Non-intrusive: purely additive — it only colors bracket glyphs and leaves all other highlighting untouched.
  • Safe: no permissions, no network, no file access.
